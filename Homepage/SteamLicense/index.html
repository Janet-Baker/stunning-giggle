Steam 许可证美化排版

这个怎么用？
  1. 打开Steam控制台
  2. 输入 clear_console
  3. 输入 licenses_print
  4. 复制屏幕上的内容(右键单击, select all, copy)
  5. 把复制的内容粘贴到上面
许可证条目太多了？

通过使用SteamCMD可以解决这个问题。

首先，你需要下载并登录。

完成之后，你可以通过输入这条命令将输出结果保存到一个文本文档里面:

steamcmd +login YOUR_ACCOUNT +licenses_print +quit > licenses.txt
